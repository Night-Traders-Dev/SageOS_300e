# SageOS Architecture: x86_64

**Status: Active Multitasking (v0.7.3 Standard)**

SageOS on x86_64 targets modern 64-bit Intel and AMD processors. It supports booting via UEFI and Multiboot2.

## Target Hardware
- **Standard PC**: Generic UEFI-compliant systems.
- **QEMU `q35` machine**: Primary emulation target.
- **Lenovo 300e Chromebook**: Specific hardware optimization target.

## Execution Environment
- **Multitasking**: Full kernel-level multitasking enabled.
- **Preemption**: Cooperative preemption via interpreted bytecode polling.
- **Isolation**: Rings 0/3 (managed via SGVM and experimental hardware paging).

## Boot Process
1.  **UEFI**: Firmware loads `BOOTX64.EFI`.
2.  **Multiboot2**: Used for Legacy/GRUB compatibility.

## Memory Management
- Uses 4-level paging (PML4) to map virtual addresses.
- **Kernel Base**: `0xFFFFFFFF80000000`.

## Drivers
- **UART**: Legacy 16550 UART (COM1) at `0x3F8`.
- **Interrupts**: APIC (Advanced Programmable Interrupt Controller) and IOAPIC.
- **VGA/Framebuffer**: Basic VBE or UEFI GOP (Graphics Output Protocol).

## Build Pipeline
Supports emission of both flat ELF binaries and UEFI-compatible PE files.
