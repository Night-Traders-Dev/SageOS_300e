# SageOS Architecture: RISC-V 64 (RV64)

SageOS on RV64 targets the open-standard RISC-V architecture, specifically the 64-bit variant with GC (General-purpose + Compressed) extensions.

## Target Hardware
- **QEMU `virt` machine**: Primary target.
- **Orange Pi RV 2**: Primary physical target (StarFive JH7110).

## Execution Environment
- **M-Mode**: Machine Mode (Firmware like OpenSBI).
- **S-Mode**: Supervisor Mode (SageOS Kernel).
- **U-Mode**: User Mode.

## Memory Mapping
- Uses Sv39 or Sv48 paging.
- **Kernel Base**: `0xFFFFFFFF80000000`.
- **Physical Load Address**:
  - Virt: `0x80200000` (standard for OpenSBI).
  - Orange Pi RV 2: `0x40000000`.

## Drivers
- **UART**: 16550 UART (compatible with QEMU virt and JH7110).
  - Virt Base: `0x10000000`.
  - JH7110 Base: `0x12030000`.
- **Interrupts**: PLIC (Platform-Level Interrupt Controller) and CLINT (Core Local Interruptor).

## Build Pipeline
Generates relocatable ELF binaries designed to be loaded by a Supervisor Binary Interface (SBI) implementation.
